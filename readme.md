<h1 align="center">Подготовка к зачету</h1>

<h5>подготовка</h5>
<p>Для начала вам понадобиться <a href="https://python.org">python</a> его можно установить на <a href="https://python.org">официальном сайте</a>
скачивайте с "добавить в PATH"</p>

<h5>Запуск</h5>
<p>Для запуска программы нужно выполнить следующую команду
<code>python main.py [-en] [--n]
-en    меняет язык, с которого нужно переводить на английский
--n    устанавливает количество предлогаемых вариантов
примеры команд:
python main.py
python main.py -en --n=4
</code>
</p>
